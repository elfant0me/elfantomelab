# Glances

Outil léger de monitoring système avec interface web, permettant de visualiser l’état global du serveur en temps réel.

## Description

Glances est un outil de supervision tout-en-un qui affiche l’utilisation :

- CPU
- mémoire
- stockage
- réseau
- processus
- conteneurs Docker

Il est idéal pour un homelab ou un petit serveur personnel.

## Accès

- Web UI : http://localhost:61208

## Fichiers

- `docker-compose.yml`
- `.env.example`
