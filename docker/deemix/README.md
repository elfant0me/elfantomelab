# Deemix

Application permettant de télécharger de la musique depuis Deezer en haute qualité.

## Description

Deemix est un outil self-hosted permettant de récupérer musique et playlists avec métadonnées complètes (tags, pochettes, etc.).

## Accès

- Web UI : http://localhost:6595

## Fichiers

- `docker-compose.yml`
- `.env.example`

## Configuration

```bash
cp .env.example .env
