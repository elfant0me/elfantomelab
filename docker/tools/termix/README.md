# Termix

Interface web légère pour accéder à un terminal depuis le navigateur.

## Description

Termix permet d’exposer un terminal web simple et pratique dans un homelab, utile pour un accès rapide à certaines tâches d’administration ou d’outillage.

## Accès

- Web UI : http://localhost:8020

## Fichiers

- `docker-compose.yml`
- `.env.example`

## Configuration

```bash
cp .env.example .env
