# Notifiarr

Service de notifications et d’intégration pour l’écosystème Servarr.

## Description

Notifiarr permet de centraliser les notifications et automatisations pour :

- Sonarr
- Radarr
- Prowlarr
- Plex / Jellyfin
- Monitoring

Il peut envoyer des notifications via Discord et autres services.

## Accès

- Web UI : http://localhost:5454

## Fichiers

- `docker-compose.yml`
- `.env.example`
