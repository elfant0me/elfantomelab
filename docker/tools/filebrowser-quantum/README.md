# File Browser Quantum

Gestionnaire de fichiers web léger et rapide, utilisé ici dans une version optimisée pour un homelab.

## Description

Cette stack utilise l’image `gtstef/filebrowser:stable-slim`, une variante légère de File Browser, pour naviguer dans les fichiers du serveur via une interface web simple.

## Accès

- Web UI : http://localhost:8147

## Fichiers

- `docker-compose.yml`
- `.env.example`

## Configuration

Copier le fichier d’exemple :

```bash
cp .env.example .env
```
