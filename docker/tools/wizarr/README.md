# Wizarr

Outil d’invitation et d’onboarding pour les utilisateurs de services multimédias.

## Description

Wizarr simplifie l’ajout de nouveaux utilisateurs à des services comme Jellyfin ou Emby grâce à une interface guidée.

Fonctionnalités :

- invitations via lien
- onboarding simplifié
- aide à la création de compte
- meilleure expérience pour famille et amis

## Accès

- Web UI : http://localhost:5690

## Fichiers

- `docker-compose.yml`
- `.env.example`
