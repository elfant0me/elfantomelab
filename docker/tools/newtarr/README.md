# Newtarr

Outil complémentaire pour automatiser et améliorer la gestion des téléchargements dans un environnement Servarr.

## Description

Newtarr agit comme un service additionnel pour Sonarr / Radarr afin d’optimiser :

- la recherche de contenu
- la gestion des téléchargements
- les automatisations avancées

## Accès

- Web UI : http://localhost:9705

## Fichiers

- `docker-compose.yml`
- `.env.example`

## Configuration

```bash
cp .env.example .env
