# Navidrome

Serveur de streaming musical self-hosted, compatible Subsonic.

## Description

Navidrome permet de gérer et écouter une bibliothèque musicale personnelle avec une interface web moderne et des applications mobiles compatibles.

Fonctionnalités :

- streaming audio
- gestion des albums / artistes
- playlists
- support Last.fm (scrobbling)
- transcodage audio

## Accès

- Web UI : http://localhost:4533

## Fichiers

- `docker-compose.yml`
- `.env.example`
