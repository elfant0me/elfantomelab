# Jellyfin
Serveur multimédia open-source pour organiser et diffuser films, séries et musique.

## Description

Jellyfin est un media server **100% gratuit et open-source**, fork de Emby, permettant de gérer et streamer du contenu sans abonnement ni compte externe.

Il fonctionne sur un modèle client-serveur et supporte de nombreux appareils (web, Android, iOS, TV, etc.).

## Accès

- Web : http://localhost:8096
