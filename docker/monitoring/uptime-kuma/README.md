# Uptime Kuma

Outil de monitoring permettant de surveiller la disponibilité des services du homelab.

## Description

Uptime Kuma permet de suivre l’état de différents services via une interface web simple et moderne.

Fonctionnalités :

- surveillance HTTP / HTTPS
- ping
- surveillance ports TCP
- notifications
- pages de statut
- historique d’incidents

## Accès

- Web UI : http://localhost:3001

## Fichiers

- `docker-compose.yml`
- `.env.example`

## Configuration

```bash
cp .env.example .env
```
