# Maintainerr

Outil d’automatisation pour gérer et nettoyer les médias dans un homelab.

## Description

Maintainerr permet de définir des règles automatiques pour :

- supprimer du contenu non regardé
- gérer l’espace disque
- nettoyer les bibliothèques médias
- automatiser la maintenance des serveurs multimédia

Il s’intègre parfaitement avec :

- Jellyfin
- Emby
- Sonarr / Radarr

## Accès

- Web UI : http://localhost:6246

## Fichiers

- `docker-compose.yml`
- `.env.example`
